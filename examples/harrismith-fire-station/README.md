# Harrismith Fire Station Reference Example

The Harrismith Fire Station is the primary synthetic project used to demonstrate
the Autodesk BIM workflow from site design through asset handover.

## Intended learning narrative

1. Establish the site and contextual information.
2. Explore one or more concept options.
3. Develop the selected design in Revit.
4. publish controlled information to the common data environment.
5. Coordinate discipline models.
6. Record and resolve issues.
7. Demonstrate reviews, RFIs, packages, or transmittals.
8. Prepare selected building assets for handover.

## Automation approach

The example deliberately uses a hybrid approach:

- Forma Site Design interface where Beta API automation creates excessive
  friction.
- Autodesk APS/Forma MCP for focused cloud and information-management actions.
- Revit MCP for model authoring and inspection.
- Native Autodesk coordination engines rather than recreating clash detection.
- Stable APIs for issues, reviews, files, and handover wherever practical.

## Public data policy

Only synthetic examples and sanitised identifiers may be committed here.
